# nrm

nrm 是一个npm源管理器，可以快速的切换npm源。

# 安装

npm install -g nrm
1

# 使用

nrm ls 可以查看当前可选的源

nrm ls
* npm ---- https://registry.npmjs.org/  # 当前使用的源
  cnpm --- http://r.cnpmjs.org/
  taobao - https://registry.npm.taobao.org/
  nj ----- https://registry.nodejitsu.com/
  npmMirror  https://skimdb.npmjs.com/registry/
  edunpm - http://registry.enpmjs.org/
1
2
3
4
5
6
7

带* 的为当前使用的源。

# 切换

npm 源管理器最核心的功能便是帮助我们快速的切换源。比如，我们需要切换到 taobao 的源地址，只需要 nrm use taobao，后续你运行 npm 相关命令，都会使用 taobao 源。

nrm use [name]
1

# 增加

增加定制的源:

nrm add [name] [url]
1
  • name: 源名称;
  • url: 源路径;

# 删除

nrm del [name]
1

# 测试源速度

nrm test [name]
1
