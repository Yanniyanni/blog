    《前端主流框架介绍》

    2020-04-10 上次更新时间：8/4/2020, 8:24:44 PM 0 工具类

    # 流行js框架

    # vue

    介绍：中小型公司使用较多。比较简单容易上手。目前稳定版本还是vue2.x，vue3.0还没正式出。

    官网

    # react

    介绍：大型公司，复杂系统用的较多。优点纯js、逻辑清晰、代码可维护性和可阅读性高。目前版本16.x了。

    官网

    # angular

    介绍：angularJS指代angular1.0版本，后来2.x之后就直接改为angular，2.x 和 1.x 版本区别巨大。几乎不能兼容升级。所以1.x版本只有一些老项目才会存在。

    官网

    # 流行移动端UI框架

    # Mint UI

    介绍：饿了么出品、搭配Vue使用。

    官网

    # Vant

    介绍：有赞出品，

    官网

    # Mui

    介绍：HCloud出品，号称最接近原生体验

    官网

    # Vux

    介绍：个人开发，但之前很火，现在不维护了，但用的人也还挺多

    官网

    # 流行PC端UI框架

    # Boostrap

    介绍：常用于门户网站之类的，需要响应式的页面（现在都少用了，因为下面的框架自带响应式）

    官网

    # ElementUI

    介绍：饿了么出品，目前很流行，搭配Vue使用。

    官网

    # AntDesign

    介绍：蚂蚁金服出品，也很流行，搭配React使用

    官网

    # Ant Design of Vue

    介绍：本来是私人团队出品，但后来被收购了，搭配Vue使用。

    官网

    # ANT DESIGN PRO

    介绍：使用AntDesign为基础写的一套开源的管理系统，适合中大型系统。

    官网

    # 流行跨端框架

    也就是一套代码可以打包出适配不同平台的代码。像H5、Android、Ios、微信小程序、百度小程序、支付宝小程序等。

    # uni-app

    介绍：HCload出品，号称跨7端，项目复杂的话坑很多

    官网

    # Taro

    介绍：京东出品，支持用 React 的语法开发。

    官网

    # react native

    介绍：使用react技术栈一般会用它，较流行。

    官网

    # weex

    介绍：阿里出品。

    官网

    # flutter

    介绍：谷歌出品。

    官网

    # Chameleon 变色龙

    介绍：滴滴出品

    官网

    # 总结

    以上所说的框架，其实都只是为了方便开发，提高效率而已。只要认真看文档就可以使用了。真正需要掌握的是原生JS。
