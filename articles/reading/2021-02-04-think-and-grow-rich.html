    # 思想决定一切，心想才能事成。

    一个人要获得成功，所需要的只是一个正确的观念。

    有成功意识的人一定会成功。轻易让自己变成失败意识者，就会失败。

    人们心中所想像和坚信的任何事物，均可实现。

    成功的阶梯，越到顶端越不挤。

    # 走向财富的十三步

    # 1. 欲望 - 所有成就的出发点，是走向财富的第一步！

    只有具有“财富意识“，才能积累财富。

    贫穷与富有，都是思想的产物。向生命要求的越多，生命带给你的也越丰富。

    把欲望变成财富必备步骤：

    1. 确定真正祈求的财务数量目标，仅说“我要很多钱”是不够的，数目一定要明确。
    2. 为了达到目标，要确定自己有决心付出些什么代价
    3. 确定一个具体的日期，决心何时拥有所企求的目标
    4. 拟定一个实现欲望的明企鹅计划，并且不论你是否已有准备，要立即开始将计划付诸行动
    5. 将要得到的财富数量目标，达到目标的期限以及为达到目标所愿付出的代价，以及如何取得这些财富的行动计划等，都简明扼要的写下来。并写一份督促自己的誓词声明。
    6. 把声明每天大声朗读两边，早起和入睡前。在读声明时，要想象到，感觉自己已经拥有了这笔财富。

    # 2. 信心 - 想象和相信欲望的实现，是走向财富的第二步！

    信心是“永恒的特效药”，它赋予思想以生命、力量和行动。

    如果你认为被击败了，那你必定被击败。

    如果你认为不敢，那你必然不敢。

    如果你想胜利，但你认为你可能获胜，那么你就不可能得到胜利。

    如果你认为你会失败，那你就已经失败了。

    因为在这个世界里，成功是从一个人的意志开始的，而意志则全靠精神。

    如果你认为你比别人优越，那你一定比别人优越。

    在你得到奖励之前，你必须对自己有自信。

    人生的战斗，并不是始终有利于较强的一方，但最终的胜利是属于自己认为有力量的人。

    # 3. 自我暗示 - 是影响潜意识的媒介，是走向财富的第三步！

    自我暗示是思想意识和外部行动之间沟通的媒介。

    让潜意识相信你必定会得到你所想像的财富，这笔财富已在等待你去领取。这样，你的潜意识一定会向你提出获得这笔财富的实际计划。

    当这些计划出现时，可能向灵感似的在你的心头闪过。要尊重这种灵感，并在接到这种灵感后，立即采取行动。

    每个逆境，每次失败，和每种烦恼，都带有等价或更大利益的种子。

    # 4. 专业知识 - 个人经验与观察，是走向财富的第四步！

    学会组织和利用知识去赚钱。

    知道如何“购买“知识的人是一定会获得报酬的！

    # 5. 想象力 - 是心灵与智慧的工厂，是走向财富的第五步！

    创意是所有财富的出发点。

    你所需要的机会，随时都在你的想像中等你。

    创意是可以转换为财富的。

    # 6. 计划 - 让欲望结晶成为行动，是走向财富的第六步！

    如果你的计划失败了，就用新的计划。 你的成就不可能大于你的计划的正确性。

    制定实际可行计划的方法：

    1. 你必须吧自己所需要的人结合在一起，以创造和实现你积累财富的计划。
    2. 在形成“智囊团”之前，你要决定对这个组织中的每个人付出一些什么代价，才能换取他们的合作。没有人会永远为你工作而不要报酬的。
    3. 在作出的安排中，至少每周要有两次与你“智囊团”里的人员会面。可能的话，会面次数可以增多。
    4. 你和“智囊团”中的每一个人都应保持完全和谐的关系。

    你知道自己的价值吗？

    # 7. 决心 - 克服拖沓和犹豫不决，是走向财富的第七步！

    做一个沉默的行动者。

    舆论是世界上最廉价的东西，不要被别人的意见所左右。

    只要你知道自己所要的是什么，通常都可以得到。

    # 8. 毅力 - 坚定信心，不断努力，是走向财富的第八步！

    缺乏毅力是失败者的共同弱点。

    毅力是一种心理状态，它是能够培养的。

    毅力需要有明确的动机作为基础。这些动机包括：明确的目标、欲望、自励、明确的计划、落实计划、合作精神、意志、习惯。

    世界上所有的机遇，都是靠自己的努力得到的。这种机遇需要有明确的目标和顽强的毅力。

    # 9. 智囊 - 集体的知识与智慧，是走向财富的第九步！

    依靠集体的知识和智慧。

    # 10. 神秘的性 - 性欲的转移与升华，是走向财富的第十步！

    性欲望是人类欲望中最有力量的一项。将性欲的力量转移到肉体意外行动，可以提升一个人到伟大成就的阶段。

    个人“吸引力”，其实就是性能量。高度性感的人通常拥有充分的吸引力。

    这种力量可以通过这些方式与其他人加以沟通：

    • 握手
    • 声音的音调
    • 身体的姿态与举止
    • 思想交流
    • 服饰仪容

    # 11. 潜意识 - 潜意识的苏醒与迸发，是走向财富的第十一步！

    潜意识向无穷智慧吸取力量，自动将一个人的欲望，转变为等价的物质。

    潜意识是会懒惰的。

    利用积极情绪。欲望、信心、爱、性、热情、依恋、希望。

    清除消极情绪。恐惧、嫉妒、怨恨、报复、贪婪、迷信、愤怒。

    # 12. 头脑 - 头脑是思想的发射站和接收站，应用头脑是走向财富的第十二步！

    成功的阶梯，越到顶端越不挤。

    # 13. 第六感 - 打开了智慧庙堂之门，是走向财富的第十三步！

    有时候“第六感”能带来奇迹。

    让伟大的人塑造你的人生。

    用自我暗示建立个性。
