      串

      上次更新时间：4/5/2021, 1:08:16 PM 0

      串（String）是由零个或多个字符组成的有限序列，又名叫字符串

      串的比较是通过组成串的字符之间的编码来进行的。（使用标准的ASCLL编码）

      # 抽象数据类型定义

      • ADT 串（String）

        • Data
          • 串的元素仅由字符组成，相邻元素具有前驱和后继关系
        • Operation
          • Clear(): 清空串。
          • Empty(): 串是否为空
          • length(): 返回串的元素个数。
          • Compare(S, T): 比较两个串，S > T,返回值 > 0，S = T，返回值 = 0，S < T，返回值 < 0。
          • Concat(T, S1, S2): 用 T，返回 S1 和 S2 连接的新串。
          • SubString(Sub, S, pos, len): 用 Sub 返回串 S 的第 pos个字符串起长度为 len 的子串
          • Index(S, T, pos):
          • Replace(S, T, V): 用 V 替换掉 S 中所有与 T 相等的不重叠子串
          • Insert(S, pos, T): 在 S 的第 pos 个字符前插入 T
          • Delete(S, pos, len): 在 S 中删除第 pos 个字符起长度为 len 的子串

      • endADT

      # 存储结构

      串也可以有两种存储结构，它们与线性表存储结构类似

      • 顺序存储结构：比较灵活，性能较好，建议使用
      • 链式存储结构：相比下没有顺序结构那么好。

      # 串的实现

      JavaScript 中有专门针对字符串提供的一系列函数，如 subString，replace 等等，这里不再赘述。
