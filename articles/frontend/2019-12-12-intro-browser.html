              《五大主流浏览器及四大内核》

              2019-12-12 上次更新时间：8/4/2020, 8:24:44 PM 0 浏览器

              # 国内常用浏览器

              IE、Firefox、QQ浏览器、Safari、Opera、Google Chrome、百度浏览器、搜狗浏览器、猎豹浏览器、360浏览器、UC浏览器、遨游浏览器、世界之窗浏览器

              # 五大主流浏览器

              • IE
              • Firefox
              • Google Chrome
              • Safari
              • Opera。

              # 四大内核

              • Trident ['traɪd(ə)nt]内核，也称IE内核。
              • Webkit内核。
              • Presto ['prestəʊ] 内核。
              • Gecko ['gekəʊ]内核。

              # 各浏览器使用内核情况

              浏览器名称 使用内核 备注
              IE Trident内核，俗称IE内核
              Opera Blink内核 最初是Presto内核，再到Webkit内核，现在是Blink内核
              Safari Webkit内核
              Firefox Gecko内核，俗称Firefox内核
              Chrome Blink内核 以前是Webkit内核
              遨游 Trident（兼容模式）+Webkit（高速模式）
              搜狗 Trident（兼容模式）+Webkit（高速模式）
              QQ浏览器 Trident（兼容模式）+Webkit（高速模式）
              世界之窗 IE内核
              百度浏览器 IE内核
              360浏览器 IE+Chrome双内核
              2345浏览器 IE+Chrome双内核 以前是IE内核
              猎豹 IE+Chrome双内核

              # 各浏览器发布时间

              浏览器名称 诞生时间 所属公司 备注
              IE 1994年 微软
              Opera 1995年 Opera Software ASA（挪威） 2016年2月，被360 和昆仑万维收购，2018年6月Opera上市
              Safari 2003年 苹果公司 是最早使用webkit内核的浏览器
              Firefox 2004年 Mozilla公司
              Chrome 2008年12月 google
              遨游 2004年 傲游公司 最早可以追溯到2010年，但在2004年正式更名为傲游
              世界之窗 2004年3月 北京超卓科技有限公司（原凤凰工作室）
              360浏览器 2008年4月 北京奇虎科技有限公司（奇虎360）
              2345浏览器 2010年12月 上海二三四五网络科技有限公司
              QQ浏览器 2010年（ios版） 腾讯科技（深圳）有限公司
              百度浏览器 2011年6月 百度 2019年9月30日上午，百度宣布，将从现在起正式停止百度PC浏览器功能
              猎豹 2012年5月 猎豹移动公司（原金山网络）
              搜狗 2012年11月 搜狗公司 原为搜狐公司旗下子公司，2010年拆分为独立公司
